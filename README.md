​[May Food] - Online Food Ordering System
​Project Overview
​This is an academic web-based application designed to provide a user-friendly experience for ordering food online. 
The project focuses on creating a seamless interface for customers while providing robust management tools for administrators.

​Technologies Used
​Frontend: HTML5, CSS3, JavaScript (Focus on aesthetic and responsive design)
​Backend: PHP
​Database: MySQL (Managed via XAMPP)
​Version Control: Git & GitHub

Key Features (CRUD Operations)
​User Authentication: Secure Login and Registration system implemented using PHP Sessions.
​Product Management (CRUD):
​Create: Administrators can add new food items to the menu.
​Read: Customers can browse through available dishes dynamically.
​Update: Easy modification of food prices, details, and availability.
​Delete: Removal of outdated or unavailable items from the database.
​Order Tracking: Real-time order processing where administrators can view and manage customer requests.

​How to Run
​Clone this repository to your local machine.
​Move the project folder to C:/xampp/htdocs/.
​Import the provided .sql file in the database/ folder into your phpMyAdmin.
​Open your browser and navigate to localhost/FoodOrderingSystem.
